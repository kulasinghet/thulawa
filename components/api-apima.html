Api Apima - Live in Concert

නීති පීඨ ශිෂ්‍යයන්ගේ සුබසාධනය අරමුණු කරගත් “අපි අපිම” අරමුදල නිර්මාණය වන්නේ “අත් ගොඩක - හිත් බැම්ම” යන තේමාවෙන් යුක්තව 2020 වසරේ දී පැවැත්වූ “අපි අපිම” ප්‍රසංගයට සමගාමීව ය. නීති පීඨ විද්‍යාර්ථීන්ගේ කලා කුසලතා වලට මනා තෝතැන්නක් එළිපෙහෙළි කරමින් සහ අධ්‍යයන පීඩනයෙන් තරමක අස්වැසිල්ලක් ලබනු වස් මෙම ප්‍රසංගය උපකාරී වනවා නො අනුමාන ය. දිවයිනේ සතර දෙසින් නා නා විධ තරාතිරමෙන් යුක්තව අගනුවර නීති වනපොත් කරන්නට පැමිණෙන විද්‍යාර්ථීන්ට ඔවුන්ගේ පෞද්ගලික මට්ටමින් පැන නැගෙන ගැටළු විසඳා ගැනීම සෑහෙන පමණ දුෂ්කර කර්තව්‍යයක් බව නොකිව මනා ය. එකී දුෂ්කර ම කාර්තුවලට ශක්තිමත්ව මුහුණ දීම සඳහා සහෝදර විද්‍යාර්ථීන්ට සවියක් වීමේ සද්භාවී චේතනාවෙන් “අපි අපිම” ප්‍රසංගයෙන් ලැබෙන මූල්‍යමය ලාභය උපයෝගී කරගෙන “අපි අපිම” අරමුදල නිර්මාණය කර තිබේ. මෙම අරමුදල නීති පීඨයේ ඉදිරි පරපුර සඳහා වර්ධනය කරමින් සුරක්ෂිත කර ගැනීම පීඨයේ තත්කාලීන සහෝදරත්වයට පැවරෙන වගකීමක් වන අතර මෙහි ක්‍රියාකාරීත්වය පිළිබඳව මෙහි බාගත කරගත හැකි හැදින්වීමේ ලේඛනයේ සදහන් වනු ඇත.
